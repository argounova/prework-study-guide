Prework Study Guide

HTML

  • HTML stands for Hypertext Markup Language
  • HTML is responsible for creating the structure of a webpage
  • Common webpage elements include text, images, header, body and list
  • HTML contains Metadata which is data about the webpage not typically seen by the user
  • The body of an HTML document includes content the user actually sees

CSS

  • CSS stands for Cascading Style Sheets
  • How webpage content is presented to users is defined through CSS
  • Three CSS methods exist: inline, internal and external (this being the most common)
  • Separation of Concerns refers to separating code in different files and is typical when using CSS
  • Utilizes "Do Not Repeat Yourself (DRY)" coding principles

Git

  • Version control software that works with a Command Line Interface
  • Git commands always start with "git"
  • Ability to save and track versions of code, initiate a repo and add code to it
  • Tracks changes locally and works with cloud based GitHub for global access

JavaScript

  • Powerful programming language that can add interactivity to a website
  • Use the "var" keyword to declare a variable
  • Variables are named and contain strings
